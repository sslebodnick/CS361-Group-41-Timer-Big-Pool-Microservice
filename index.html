⏱️ Timer / Stopwatch Microservice

Welcome to the Timer Microservice! This service allows you to start, stop, and track timers programmatically or through the web interface.

Features

  • ✅ Start timers with optional labels
  • ✅ Stop timers and get elapsed time in multiple formats
  • ✅ View all running and stopped timers
  • ✅ Reset timers to start over
  • ✅ Delete timers when no longer needed
  • ✅ RESTful API for programmatic access

Quick Start

Use the navigation bar above to:

  • Start Timer: Create new timers with custom labels
  • View Timers: See all timers and their elapsed times

API Example

Start a timer:
POST http://localhost:3001/timers/start
{ "label": "My Task" }

Stop a timer:
POST http://localhost:3001/timers/{id}/stop

Get timer status:
GET http://localhost:3001/timers/{id}
